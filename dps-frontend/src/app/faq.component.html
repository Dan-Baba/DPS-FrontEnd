How to Volunteer
To volunteer for a job you must first have a registered account with a valid username and password. If you do not have one, you can create one by clicking on the register/login button on the top right corner of the screen and click the register button. Make sure you are logged in to your account and then find the job that you wish to volunteer for. Then click on the details button for that job. If the job has open positions then you will see a volunteer button. Now all you have to do is click on the button to be volunteered for that job.
How to use chat
To use chat simply click on the chat icon(mail symbol) on the top right corner of your screen while being logged in. You are now able to compose and read messages in the site-wide chat.
How to contact an admin
If you wish to contact an admin you can do so by using the chat feature.
Old event not showing up?
If an event took place before the current date, then it will not be displayed.
